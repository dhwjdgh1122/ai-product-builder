
# 프로젝트 개요

이 프로젝트는 사용자가 버튼 하나로 쉽게 로또 번호를 생성, 확인하고 기록을 관리할 수 있는 웹 애플리케이션입니다. 깔끔하고 현대적인 디자인을 적용하여 사용자 경험을 향상시키는 데 중점을 두었습니다.

# 디자인 및 기능

*   **메인 화면:**
    *   **헤더:** "로또 번호 생성기"라는 명확한 제목과 다크/라이트 모드 전환 스위치를 표시합니다.
    *   **번호 디스플레이:** 생성된 6개의 로또 번호가 세련된 원형 디자인 안에 표시됩니다.
    *   **생성 버튼:** 클릭 시 새로운 번호를 생성하고, 생성된 번호는 하단 기록에 추가됩니다.
    *   **히스토리:** 생성된 번호 기록이 최신순으로 정렬되어 목록 형태로 표시됩니다.
*   **테마:**
    *   시스템 설정을 감지하여 자동으로 다크/라이트 모드를 적용하며, 사용자가 수동으로 변경하고 선택을 저장할 수 있습니다.
*   **전체 디자인:**
    *   CSS 변수를 활용하여 다크/라이트 모드에 유연하게 대응하는 디자인 시스템을 갖췄습니다.
    *   Poppins 폰트와 입체적인 그림자 효과를 사용하여 현대적인 UI를 구현했습니다.

# 현재 구현 계획: Formspree를 이용한 제휴 문의 폼 추가

1.  **HTML 수정:**
    *   `index.html`의 `main` 태그 내부에 제휴 문의를 위한 새로운 섹션(`contact-card`)을 추가합니다.
    *   해당 섹션 안에 Formspree 엔드포인트(`https://formspree.io/f/mreakeqg`)로 연결되는 폼을 생성합니다. 폼에는 '이름', '이메일', '문의 내용' 필드와 '보내기' 버튼이 포함됩니다.
2.  **CSS 수정:**
    *   `style.css`에 `.contact-card` 및 내부 폼 요소(input, textarea, button)에 대한 스타일을 추가합니다. 기존 디자인 시스템(카드 스타일, 색상 변수 등)과 일관성을 유지하여 다크/라이트 모드에서도 자연스럽게 보이도록 합니다.
3.  **JavaScript 수정:**
    *   `main.js`에 폼 제출(submit) 이벤트를 감지하는 로직을 추가합니다.
    *   AJAX(`fetch` API)를 사용하여 페이지 새로고침 없이 폼 데이터를 비동기적으로 제출합니다.
    *   제출 성공 시, 사용자에게 성공 메시지를 보여주고 폼을 초기화합니다. 실패 시에는 에러 메시지를 표시합니다.
4.  **배포:**
    *   모든 코드 변경사항을 `git`으로 커밋하고 원격 저장소에 푸시하여 최종 배포를 완료합니다.
